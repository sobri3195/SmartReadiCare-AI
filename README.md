# SmartReadiCare AI Frontend MVP

Frontend MVP sudah di-upgrade ke **React.js + TailwindCSS + shadcn-style components + animation**.

## Stack
- React.js (Vite)
- TailwindCSS
- shadcn-style reusable UI components (`src/components/ui`)
- Framer Motion + tailwindcss-animate

## Jalankan
```bash
npm install
npm run dev
```

Buka `http://localhost:5173`.

## Struktur
- `src/App.jsx` — dashboard MVP (mobile-first + desktop sidebar)
- `src/components/ui` — komponen Card, RiskBadge, Button (gaya shadcn)
- `docs/smartreadicare-uiux-mvp.md` — konsep UI/UX detail
